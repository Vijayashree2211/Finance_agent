# Finance_agent

Finance_agent is a Python-based application that provides personalized financial advice, investment suggestions, and savings tips based on a user's age, financial goals, and expenses. It uses REST APIs and a user-friendly web interface (built with Flask and Streamlit) to recommend asset allocation across stocks, mutual funds, gold, real estate, and bonds. The application also suggests top-performing stocks and mutual funds, gives tips for optimizing savings, and can recommend travel destinations within a specified budget.

## Libraries Used

- **Flask**: For building RESTful APIs.
- **Flask-RESTx**: For API documentation and structure.
- **Flask-CORS**: To handle Cross-Origin Resource Sharing.
- **Streamlit**: For building the interactive web UI.
- **Plotly**: For data visualization (pie charts, etc.).
- **SQLAlchemy**: For database modeling and ORM.
- **yfinance**: To fetch stock market data.
- **requests**: To make HTTP requests to external APIs (including Perplexity AI).
- **python-dotenv**: For managing environment variables.
- **langchain**: For building language model-powered workflows and financial reasoning.
- **Perplexity API**: For intelligent recommendations and insights.
